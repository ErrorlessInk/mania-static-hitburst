\# Mania Static Image Hitburst



An Etterna-style last judgement image overlay for osu!mania using tosu.



\## Files

\- index.html

\- style.css

\- index.js

\- metadata.txt

\- images/

What you will get in osu! mania:

![Animated preview](preview.gif)

\## How to use

Put the overlay files in your tosu ***static*** folder
